## API

各组件属性与原生 `DOM` 标签保持一致，分别为

`Table` -> `table`

`TableHeader` -> `thead`

`TableBody` -> `tbody`

`TableFooter` -> `tfoot`

`TableRow` -> `tr`

`TableHead` -> `th`

`TableCell` -> `td`

`TableCaption` -> `caption`
