## API

Extended from `button`

| Property | Description | Type                                                                                       | Default Value |
| -------- | ----------- | ------------------------------------------------------------------------------------------ | ------------- |
| variant  | 变体类型    | `"link" \| "default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| undefined` | `undefined`   |
| size     | 大小        | `"default" \| "sm" \| "lg" \| "icon" \| undefined`                                         | `undefined`   |
